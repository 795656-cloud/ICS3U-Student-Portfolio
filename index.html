Nameplate Assignment

A creative Java Processing sketch featuring my name, Pakistan's flag, a boat, and randomly spawning fish

Overview

This is an interactive Processing sketch that displays my name "YUSHA" with creative letter designs, Pakistan's flag, a detailed boat, and colorful fish swimming randomly in the water. The "S" is transparent to symbolize remembering your roots.

Key Features

  • Custom drawn letters spelling out my name (YUSHA)
  • Pakistan's flag with accurate crescent moon and star design
  • A detailed boat with cockpit and windows floating on water
  • Seven colorful fish that spawn randomly in the water
  • The "S" is transparent to symbolize remembering your roots (the background shows through)

Challenges & Solutions

The main challenge I faced was using arcs in Java Processing and understanding how to make them go where I wanted. The solution was that I just used a Bezier curve for the letter "S" instead, which gave me much better control over the shape and positioning.

Distributed File Storage System

A scalable distributed system for storing and retrieving files with built-in redundancy and fault tolerance

Overview

This project implements a distributed file storage system inspired by Google File System (GFS). It supports storing large files across multiple servers with automatic replication for fault tolerance and efficient data retrieval.

Technologies Used

Go gRPC Docker Kubernetes Redis

Key Features

  • Automatic file chunking and distribution across multiple servers
  • Three-way replication for data redundancy
  • Master-slave architecture for metadata management
  • Heartbeat mechanism for detecting node failures
  • Automatic re-replication when nodes fail
  • Load balancing for read operations

Technical Highlights

The system uses consistent hashing to distribute files evenly across storage nodes. I implemented a custom consensus protocol for handling master node failures and ensuring metadata consistency. The system can handle concurrent reads and writes while maintaining data integrity.
